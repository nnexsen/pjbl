# Pembuatan Website Desa Sumberjo
Kelompok 4 — Tugas Sekolah  
**Rohaf Azzam Miqdad, Muhammad Denny Syahputra, Muhammad Naja Zainul Wafa, Ahmad Nabi' Syuhadak**  
Tema: Pixel art, warna hijau (#2E8B57), biru (#1F6FEB), kuning (#FFC857)  
<small>Rembang, 2025</small>

---

# Rumusan Masalah
- Bagaimana menyediakan informasi desa yang rapi & mudah diakses?
- Bagaimana mempublikasikan berita/agenda resmi dengan cepat?
- Bagaimana menampilkan data statistik desa secara ringkas?
- Bagaimana menjaga tampilan modern tapi tetap ringan?

---

# Fitur pada Website
- Hero Desa Sumberjo (logo & lokasi)
- Navigasi: Profil, Sejarah, Berita, Visi Misi
- Carousel foto kegiatan desa
- Sambutan Kepala Desa
- Statistik Desa: wilayah, agama, kewarganegaraan, pekerjaan, gender, pendidikan
- Grid berita: judul, ringkasan, tanggal, views
- Footer: alamat, sosial media, tautan terkait, penghitung pengunjung
- Search bar & dark mode (pengembangan)

---

# Proses Pembuatan
1. Perencanaan konten & struktur halaman
2. Desain wireframe + tema pixel-art
3. Implementasi HTML + CSS (Bootstrap 4.5) + JS ringan
4. Integrasi gambar/logo/ikon
5. Interaktivitas: carousel, tombol, handler dasar
6. Uji responsif & aksesibilitas dasar
7. Rencana lanjut: data dinamis (berita/statistik)

---

# Teknologi yang Digunakan
- **Frontend:** HTML5, CSS3, Bootstrap 4.5, JS vanilla
- **Font:** Poppins (Google Fonts)
- **Aset:** gambar & ikon pixel art (sumber bebas/berlisensi)
- **Hosting (opsional):** GitHub Pages / Netlify / Vercel
- **Next:** API berita/statistik, dark mode persist, search nyata

---

# Dalil Al-Qur’an (Relevansi Informasi)
- QS. An-Nahl: 90 — “Allah menyuruh berlaku adil dan berbuat kebajikan…”  
  → Transparansi & layanan publik.
- QS. Al-Hujurat: 6 — “Jika datang berita, telitilah kebenarannya…”  
  → Verifikasi berita sebelum disebar.
- Website mendukung keterbukaan, akurasi, dan kemaslahatan warga.

---

# Manfaat
- Warga mudah akses berita & agenda resmi
- Transparansi data dasar desa
- Memperkuat citra desa dan komunikasi publik
- Proyek TIK nyata untuk siswa (showcase ke guru)

---

# Kelebihan & Kekurangan
**Kelebihan**
- Tampilan modern & ringan
- Navigasi jelas, konten terstruktur
- Siap dikembangkan ke data dinamis

**Kekurangan (saat ini)**
- Data masih statis
- Search & dark mode belum penuh
- Belum ada backend untuk berita/statistik
- Aksesibilitas bisa ditingkatkan (alt, kontras)
